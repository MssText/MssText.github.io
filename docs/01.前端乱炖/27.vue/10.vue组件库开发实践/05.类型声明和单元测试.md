### 前言
一个成熟的 UI 库 `类型声明` 和 `单元测试`是必备的。

### 类型声明
众所周知，JavaScript 是一门弱类型语言，编译时不会检查参数的类型。这种特性虽然很灵活，但是可能会造成许多潜在的运行时错误。

我们自己开发的库，想提供 `TypeScript` 支持，有两种方式
- 本身项目是 使用 `TypeScript` 开发
- 添加 `*.d.ts` 类型声明文件
